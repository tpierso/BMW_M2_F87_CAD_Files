For BMW "Speedhole" style vents for the rear bumper. 

I created these so they can be 3D printed, sent to a vendor like SendCutSend, or an alternative. 

The rear bumper vent surround and vent mesh are two seperate files to give you options if you want to have different materials--e.g. I have my mesh made out of aluminum anodized black with carbon fiber surrounds. 

The vent install tool is meant to be used to install the completed vents into the bumper; however, you can also combine the rear bumper vent surround and the install tool to make a cut template as well. 

I also split the install tool into 3 seperate vents for smaller 3D printers. It has a simple dovetail so you can quickly glue it together to suit what you need.
